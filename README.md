# weather_app_finland-weather-analysis
A web application to display weather updates

## Features

- 5 day weather forecast
- Impending Weather changes
- Weather Graph
- Sunrise and sunset times
- CLoud Coverage
- Wind Speed

## Built Using

- Python
- Streamlit
- Plotly
- Python Open Weather Map API
